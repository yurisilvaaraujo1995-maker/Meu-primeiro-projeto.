Este projeto é o meu primeiro projeto online, desenvolvido como parte do meu aprendizado prático em desenvolvimento web.
O objetivo foi criar um portfólio pessoal utilizando HTML e CSS, apresentando informações sobre mim, minhas habilidades e o que aprendi durante o curso prático.

Esse projeto marca meu início na área de Front-End e Back-End Júnior, aplicando conceitos básicos de estrutura, estilização e organização de páginas web.
